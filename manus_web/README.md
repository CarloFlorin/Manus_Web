# DevOps Lernplattform

Eine umfassende, interaktive Lernplattform für DevOps mit Terminal-Brutalismus-Design.

## Features

- **13 Lernpfade**: Von Linux-Grundlagen bis zu Kubernetes-Orchestrierung
- **Terminal-Design**: Authentische Terminal-Ästhetik mit Cyan, Grün und Orange Akzenten
- **Mobile-optimiert**: Vollständig responsive für Smartphones, Tablets und Desktop
- **Cheat-Sheets**: Schnelle Referenzen für alle wichtigen Befehle
- **Wiki & Dokumentation**: Detaillierte Erklärungen zu jedem Thema
- **Praktische Projekte**: Real-World Szenarien zum Hands-On Lernen

## Lernpfade

### Fundamentale Bausteine (Woche 1-8)
- Linux-Grundlagen
- Git & Versionskontrolle
- Netzwerk-Grundlagen
- Scripting (Bash & Python)

### Kernbereiche (Woche 9-34)
- CI/CD & Automation
- Docker & Container
- Kubernetes
- Infrastructure as Code
- Cloud-Plattformen
- Monitoring & Logging

### Zusätzlich
- Praktische Projekte
- Ressourcen & Weiterführendes

## Installation

```bash
cd manus_web
pnpm install
pnpm dev
```

Die Website ist dann verfügbar unter `http://localhost:3000`

## Technologie-Stack

- **Frontend**: React 19 + Tailwind CSS 4
- **Routing**: Wouter
- **UI-Komponenten**: shadcn/ui
- **Icons**: Lucide React
- **Typografie**: JetBrains Mono, IBM Plex Mono

## Design-Philosophie

Die Plattform folgt der Terminal-Brutalismus-Ästhetik:
- Dunkler Hintergrund (#1a1b26)
- Leuchtende Akzentfarben (Cyan #7dcfff, Grün #9ece6a, Orange #ff9e64)
- Monospace-Typografie durchgehend
- Minimale aber präzise Animationen
- Terminal-Fenster-Stil für Content

## Responsive Design

- **Mobile**: Hamburger-Menü, optimierte Typografie, Touch-freundliche Buttons
- **Tablet**: Angepasste Layouts und Abstände
- **Desktop**: Vollständige Sidebar-Navigation
